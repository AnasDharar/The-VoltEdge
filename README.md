# The VoltEdge

**The VoltEdge** is a comprehensive, open-source toolkit designed for Digital Logic Design (DLD) and Electronics enthusiasts, students, and engineers. It provides interactive tools to solve, simulate, and understand complex digital circuits and calculations.

## üöÄ Features

The application currently includes the following tools:

- **K-Map Solver**: Simplify Boolean expressions using Karnaugh Maps (up to 4 variables).
- **Truth Table Generator**: Generate truth tables from Boolean expressions.
- **Base Converter**: Convert numbers between different number systems (Binary, Decimal, Octal, Hexadecimal).
- **Logic Gates Minimizer**: Minimize logic gate circuits for efficiency.
- **Resistor Color Code Calculator**: Calculate resistor values from color bands.
- **Logic Gate Simulator**: Visualize and simulate logic gate operations.
- **Educational Resources**: Blogs and guides on digital logic concepts.

## üì¶ Getting Started

Follow these steps to set up the project locally.

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1.  **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/The-VoltEdge.git
    ```

2.  **Navigate to the frontend directory**

    ```bash
    cd frontend
    ```

3.  **Install dependencies**

    ```bash
    npm install
    ```

4.  **Run the development server**

    ```bash
    npm run dev
    ```

5.  Open your browser and navigate to `http://localhost:5173`.
